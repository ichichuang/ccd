# Antigravity Skills

This directory contains standard operating procedures (SOPs) for the Antigravity Agent.

## Skill List

- **[01-build-ui-component](01-build-ui-component.md)**: Create a new UI component (Scaffold -> Style -> Verify).
- **[02-build-page-view](02-build-page-view.md)**: Assemble a full page view (Layout -> Components -> Logic Integration).
- **[03-fix-ui-issue](03-fix-ui-issue.md)**: Debug and fix UI bugs (Reproduce -> Locate -> Fix -> Verify).
- **[04-full-feature](04-full-feature.md)**: End-to-end feature implementation (API Definition -> Business Hook -> UI Integration).
- **[05-refactor-component](05-refactor-component.md)**: Extract logic and standardize styles (Extract Logic -> Hooks & Style Audit).
- **[06-style-audit](06-style-audit.md)**: Fix hardcoded styles (Find Hardcoded -> Replace with UnoCSS).

## Usage

When assigned a task, verify if it falls into one of these categories and follow the corresponding SOP.
