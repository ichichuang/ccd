# CCD Product UI Profile

This profile defines the current replaceable CCD Architectural Glass expression. Agents must apply it through registered tokens, shared primitives, and platform invariants, and must treat it as replaceable product language rather than permanent runtime architecture.

## Profile Identity

The current profile is CCD Architectural Glass: quiet, structured, premium, architectural, token-driven, accessible, and restrained.

## Replaceable Status

This profile is replaceable by an approved future governance lane. Replacement cannot weaken platform invariants, accessibility, token ownership, component priority, or runtime behavior.
