# CCD Product Language

CCD product language governs how UI communicates architecture, evidence, and operational control. Agents must use it to reject generic dashboard decoration and preserve task clarity before choosing material, density, motion, or emphasis.

## Purpose

CCD UI language communicates architecture, evidence, and operational control. Interfaces should feel precise, calm, and inspectable rather than decorative or promotional.

## Rejected Product-Language Patterns

Reject generic AI-dashboard decoration, random gradients, excessive glass, purposeless nested cards, oversized headings without information value, gratuitous hover motion, decorative animation on utility surfaces, marketing-page composition inside operational tools, and novelty that obscures architecture or evidence.

## Relationship to the Product UI Profile

The product language supplies durable product intent. The Product UI Profile translates that intent into replaceable visual expression without changing platform invariants.
