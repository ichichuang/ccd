# Accessibility

Accessibility governance applies to every user-facing UI change. Agents must preserve semantic structure, keyboard access, visible focus, announcements, contrast, non-color communication, responsive reading order, and reduced-motion behavior.

## Semantic Structure

This section requires agents to apply semantic structure rules with explicit ownership, current monorepo evidence, accessibility-safe behavior, token-driven styling, and validation before reporting completion.
