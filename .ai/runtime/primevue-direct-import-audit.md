Policy implications for later tasks:

- App-authored source should not import `primevue/*` directly except exact governed bootstrap/build/generated boundaries.
- `packages/vue-ui/src/**` may import PrimeVue internally but must not publicly re-export raw PrimeVue modules.
- Theme, PassThrough, services, global PrimeVue install, and locale application should stay in `packages/vue-primevue-adapter/**`.
