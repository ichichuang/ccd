# Final Artifacts Required at Completion

## `FINAL_GO_NO_GO.md`

Required content:

- final status: `GO`, `CONDITIONAL_GO`, or `NO_GO`;
- milestones completed;
- milestones deferred;
- blockers;
- validation summary;
- risk summary;
- reviewer decision;
- operator signoff placeholder.

## `FINAL_VALIDATION_MATRIX.md`

Required content:

- every required command;
- last run timestamp;
- result;
- evidence log path;
- notes;
- skipped checks and rationale.

## `OPERATOR_SOP.md`

Required content:

- how to rerun validation;
- how to inspect evidence;
- how to handle generated drift;
- how to continue next milestone;
- how to rollback;
- how to prepare commit with allowlist;
- explicit “do not push unless approved”.

## `NEXT_ACTIONS.md`

Required content:

- next recommended milestone;
- blocked decisions;
- deferred lanes;
- validation still needed;
- owner approvals required;
- recommended Git action if any.

## `CHANGE_SUMMARY.md`

Required content:

- changed areas;
- architectural impact;
- behavior impact;
- validation evidence links;
- known risks;
- rollback summary.

## Final `RISK_REGISTER.md`

Required content:

- all risks updated;
- closed risks marked;
- residual risks documented;
- escalation triggers.

## Final `STATUS.md`

Required content:

- final milestone state;
- final task state;
- final validation state;
- final evidence directory;
- final blockers;
- final next action;
- working tree status.
