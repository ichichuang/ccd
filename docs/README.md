- 中文文档面向人类开发者

- `docs/generated/**` 与 `.ai/generated/**` 为机器生成产物，不可手工修改

## generated 文档说明

`docs/generated/**`、`.ai/generated/**`、`.ai/governance/api-snapshots/**` 均为受治理生成产物。

刷新命令：

```bash
pnpm governance:refresh
pnpm governance:gate
```

不要手工编辑这些文件。
