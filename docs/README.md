## Active Workspace Set

```text
packages/contracts  -> public ABI: interfaces and shared types only
packages/core       -> runtime-neutral platform logic
apps/web-demo       -> browser runtime and browser adapters
apps/desktop        -> Tauri runtime and desktop adapters
root                -> orchestration-only shell
```

## Mandatory Gate

```bash
pnpm governance:gate
```

This is the single local and CI architecture gate. It runs governance asset validation, AI guard, boundary validation, runtime leak detection, API snapshot compatibility, supply-chain policy, release topology, and governance report generation.
It also validates that GitHub's remote workflow registry matches the declared workflow topology.

Use targeted commands only when debugging a failing gate:

```bash
pnpm arch:boundaries
pnpm arch:runtime
pnpm api:report
pnpm supply:check
pnpm release:governance
pnpm ai:guard
```

## Generated Artifact Rule

Generated files are reviewable architecture state. If governance, API exports, topology, or policy files change, run:

```bash
pnpm governance:gate
```

Commit resulting updates under:

- `docs/generated/**`
- `.ai/generated/**`
- `.ai/governance/api-snapshots/**`
