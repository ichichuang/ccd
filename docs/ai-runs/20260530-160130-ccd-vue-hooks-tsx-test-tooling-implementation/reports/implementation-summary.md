## Dirty Spec Decision

The pre-existing dirty `packages/vue-hooks/src/createAutoMittHook.spec.ts` partially removed `h`, but it still used a template-backed harness and a manual `import type { PropType } from 'vue'`.

Decision: replace and rename it to `packages/vue-hooks/src/createAutoMittHook.spec.tsx` so the harness is a TSX render-return component factory:

- no value import from `vue`
- no type import from `vue`
- no `h` import
- no `h(...)` call
- no template fallback
