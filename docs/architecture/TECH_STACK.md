# 🛠️ 技术栈详解

> 本文档列出模板使用的各项技术、版本及在本项目中的用途。

---

## 📦 核心技术

| 技术           | 版本    | 用途                                            |
| -------------- | ------- | ----------------------------------------------- |
| **Vue**        | ^3.5.12 | 前端框架，仅使用 Composition API + Script Setup |
| **TypeScript** | ^5.8.3  | 类型安全，严格模式                              |
| **Vite**       | ^7.3.1  | 构建工具，开发服务器，HMR                       |
| **Pinia**      | ^3.0.3  | 状态管理，替代 Vuex                             |
| **Vue Router** | ^4.4.5  | 路由管理，支持静态 + 动态路由                   |

---

## 🎨 UI 与样式

| 技术                  | 版本    | 用途                                   |
| --------------------- | ------- | -------------------------------------- |
| **PrimeVue**          | ^4.5.4  | UI 组件库，Unstyled + PassThrough 优先 |
| **@primeuix/themes**  | ^2.0.3  | PrimeVue 主题系统                      |
| **PrimeIcons**        | ^7.0.0  | PrimeVue 官方图标集                    |
| **UnoCSS**            | ^66.6.0 | 原子化 CSS，Utility-First 样式         |
| **Animate.css**       | ^4.1.1  | CSS 动画库                             |
| **OverlayScrollbars** | ^2.14.0 | 自定义滚动条                           |
| **Sass**              | ^1.97.3 | SCSS 预处理器                          |

---

## 🌐 网络与请求

| 技术      | 版本   | 用途                                |
| --------- | ------ | ----------------------------------- |
| **Alova** | ^3.3.3 | 请求库，替代 Axios，支持缓存/重试等 |

> 📌 项目禁止在业务代码中直接使用 `axios` 或 `fetch`，统一通过 `@/utils/http` 与 `useHttpRequest` 封装。
