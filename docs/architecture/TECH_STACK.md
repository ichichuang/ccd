# 🛠️ 技术栈详解

> 本文档列出 CCD 项目使用的各项技术、版本及在本项目中的用途。

---

## 📦 核心技术

| 技术           | 版本    | 用途                                            |
| -------------- | ------- | ----------------------------------------------- |
| **Vue**        | ^3.5.12 | 前端框架，仅使用 Composition API + Script Setup |
| **TypeScript** | ^5.8.3  | 类型安全，严格模式                              |
| **Vite**       | ^7.3.1  | 构建工具，开发服务器，HMR                       |
| **Pinia**      | ^3.0.3  | 状态管理，替代 Vuex                             |
| **Vue Router** | ^4.4.5  | 路由管理，支持静态 + 动态路由                   |

## 🌐 网络与请求

| 技术      | 版本   | 用途                                |
| --------- | ------ | ----------------------------------- |
| **Alova** | ^3.3.3 | 请求库，替代 Axios，支持缓存/重试等 |

> 📌 项目禁止在业务代码中直接使用 `axios` 或 `fetch`，统一通过 `@/utils/http` 与 `useHttpRequest` 封装。
