| Tooling         | Recommendation                                                           |
| --------------- | ------------------------------------------------------------------------ |
| Codex Goal mode | Recommended                                                              |

| Automations     | Not required                                                             |
| CLI/SDK         | Use repository pnpm scripts only                                         |
| Push            | Not allowed without explicit owner authorization                         |
