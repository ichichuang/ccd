# Active Run Directory Template

When executing the plan, create a new directory with a real timestamp, for example:

`docs/ai-runs/20260602-153000-ccd-public-layer-repair/`

Do not use this placeholder directory as the active evidence directory.

Expected subdirectories:

- `reports/`
- `command-logs/`
- `diffs/`
- `screenshots/`
- `artifacts/`
